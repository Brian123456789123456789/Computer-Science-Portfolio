    • HTML

    We use HTML to tell a computer how to make a website.HTML stands for Hypertext Markup Language.HyperText Markup Language, commonly referred to as HTML, is the standard markup language used to create web pages. HTML came from the internet.HTML is written in tags.

    CSS

    Cascading Style Sheets or CSS is used to control the look and feel of the content written in HTML. CSS is used to add layout to websites. CSS was first proposed by Hakon Wium Lie. CSS is written in tags.

    Tags

    A word or phrase preceded by a hash mark (#), used within a message to identify a keyword or topic. tags are the hidden keywords within a web page that define how the browser must format and display the content. You write a tag by writing a attribute in a greater than and less than symbol.

    Attributes

    An attribute either modifies the default functionality of an element type or provides functionality to certain element types unable to function correctly without them. The href Attribute. HTML links are defined with the < a > tag. The link address is specified in the href attribute
